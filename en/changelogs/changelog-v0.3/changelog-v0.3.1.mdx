<div className="flex justify-center items-center w-full -mt-14"><h3 className="text-3xl font-bold">ProxiLink MRP</h3></div>

* **BETA ProxiLink MRP:** A unique automatic RP scene addition system enters the public beta testing stage.
  * How does it work?
    * You need to enter the **⏰┃RP Scene Waiting** channel on the **Moscow City Emergency Services** server.
    * The bot tracks your activity on the server and will create an RP Scene with players who are nearby in the game. You will be automatically moved to this scene.
    * After the scene ends, the bot will also return you back.
    * Yes, there is currently a noticeable delay, but we are constantly improving the systems.
  * **The voice channel ⏰┃RP Scene Waiting is NOT IC and any game information shared there that is later used in the game will be considered metagaming.**

> These features are released in BETA mode. We invite all players to actively use the new systems, familiarize themselves with how they work, and suggest improvements or report bugs in the appropriate channels. Your feedback will help us make the final release even better!

<div className="flex justify-center items-center w-full -mt-14"><h3 className="text-3xl font-bold">ModCalls 2.0</h3></div>

**BETA ModCalls 2.0:** A completely redesigned admin call system via the `;mod` command. Requests are now processed through the MRP Department Management bot. It works approximately the same as the `!mod` command in-game, but sends calls not to the game, but to a separate channel.

* **Please note that this is an in-game command entered directly in the Roblox chat. It functions even for users with restricted chat access — you just need to type the command in the input field and send it.**
* You can also type the command as `;mod Reason for call` so that the moderator knows why you called them upon teleportation.
* Commands in Discord channels have not been removed.

> These features are released in BETA mode. We invite all players to actively use the new systems, familiarize themselves with how they work, and suggest improvements or report bugs in the appropriate channels. Your feedback will help us make the final release even better!
