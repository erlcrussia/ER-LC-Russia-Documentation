    <div className="flex justify-center items-center w-full -mt-14"><h3 className="text-3xl font-bold"><strong>E. Damage Dealing</strong></h3></div>

    * **E-1** — Shooting, punching, using knives, etc. in a moderator scene, at staff, or at any official vehicle, results in a kick.
      * **<span style={{ color: '#ff0000' }}>E-1.1</span>** <span style={{ color: '#ff0000' }}>— Players must pull over and/or yield to staff vehicles, regardless of the team.</span>
    * **E-7** — <span style={{ color: '#ff0000' }}>Civilians are allowed not to roleplay drawing a pistol. However, roleplay is mandatory when drawing rifles (e.g., M14 or LMT L123A1) and all other weapons including submachine guns.</span>
      * **<span style={{ color: '#ff0000' }}>E-7.1</span>** <span style={{ color: '#ff0000' }}></span><span style={{ color: '#ff0000' }}>— Official police officers must roleplay drawing any weapon, except for a pistol that is attached to the duty belt (holster).</span> <span style={{ color: '#00be00' }}>Every player, regardless of their role, must roleplay drawing any weapon, whether it's a pistol, rifle, or submachine gun.</span>
      * <span style={{ color: '#00be00' }}>Roleplay is allowed both in the "game-chat" of the emergency services Discord channel and in the voice channel you are in (even if you are alone in it).</span>
      * <span style={{ color: '#00be00' }}>When using a voice channel, a video recording in which your words are CLEARLY and loudly audible is mandatory.</span>
      * <span style={{ color: '#00be00' }}>Roleplay must be voiced strictly before shooting begins and must mention the specific type or name of the weapon, as well as the fact of bringing it to combat readiness (simply "drew weapon" is not enough).</span>
      * **<span style={{ color: '#00be00' }}>Important clarification:</span>** <span style={{ color: '#00be00' }}></span><span style={{ color: '#00be00' }}>roleplay is only required when drawing a weapon; when you put it away, roleplaying the action is not mandatory.</span>
