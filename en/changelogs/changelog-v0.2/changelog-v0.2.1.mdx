<div className="flex justify-center items-center w-full text-center">
  A major infrastructure and project regulation update is officially live! We have completely rewritten the community rules and implemented an updated tech stack to make your interaction with the server more comfortable, and system operations more intuitive and faster.
</div>

***

* **Gameplay and Guides:**
  * We have completely reworked the formatting of the [RP Terms](https://wiki.erlcrussia.xyz/documentation-moscow-roleplay/general-rules/rp-terms) section. Information is now absorbed faster, and all unnecessary and outdated content has been removed.
  * The player guide has become cleaner and more concise — only what is truly needed for quality gameplay.
* **"Moscow RolePlay" Bot Update:**
  * The ticket system has undergone a complete software overhaul. What changed:
    * The system is now faster and more stable.
    * Added essential tools for working with requests:
      * `$take` — assign a ticket to yourself.
      * `$release` — remove responsibility from the current moderator.
      * `$open` / `$close` — manage access to the conversation.
    * All bot messages and interfaces have been migrated to V2 components (improved buttons, menus, and attachments).
* **ER:LC Russia API Launch:**
  * We are excited to announce the launch of our own API, which will open up new possibilities for integrations and third-party developers:
  * Detailed method descriptions are available here — [**https://apidocs.erlcrussia.xyz/**](https://apidocs.erlcrussia.xyz/)
  * Keys are issued through tickets in the **"**[**API Support**](https://discord.com/channels/1331331077553262765/1416175944829243422)**"** category.
  * Be sure to review the requirements in the documentation or clarify details when creating a ticket.
* **Information Channels and Interface:**
  * Visual design has been completely updated across all key sections:
    * ER:LC information menu.
    * Discord information and rules menu.
  * The unwanted mention protection system has been updated to the latest version for a more comfortable communication experience.
* **Staff Management:**
  * The staff promotion and reporting system has been moved to a specialized "Department Management" bot. This will allow more effective tracking of employee progress and automate routine tasks.
