* **Обновление пункта A-8 правил лидеров:**
  * **A-8** — «День блата»: В первые 24 часа с момента назначения на пост, лидер имеет право свободно распределять любые ранги внутри фракции (назначение заместителей, глав подразделений и т.д.).
    * **A-8.1** — В этот период разрешается принимать неограниченное количество новых сотрудников на любые должности без официального принятия (**правило** **B-7**).
    * **<span style={{ color: '#ff0000' }}>A-8.2</span>** — <span style={{ color: '#ff0000' }}>Для стандартного увольнения или понижения игрока по-прежнему требуется веская законная причина (РП-составляющая (</span>**<span style={{ color: '#ff0000' }}>B-7</span>**<span style={{ color: '#ff0000' }}>)).</span>
    * **<span style={{ color: '#ff0000' }}>A-8.3</span>** — <span style={{ color: '#ff0000' }}>В «День блата» лидер может получить специальное разрешение от Куратора организации на увольнение сотрудников без объяснения законных причин (например, для полной расчистки состава под новую команду).</span>
    * **<span style={{ color: '#00be00' }}>A-8.2</span>** — <span style={{ color: '#00be00' }}>В «День блата» лидер имеет право свободно понижать и повышать любых сотрудников в должности без объяснения причин. Однако для увольнения игрока по-прежнему требуется веская законная причина (РП-причина).</span>
    * **<span style={{ color: '#00be00' }}>A-8.3</span>** — <span style={{ color: '#00be00' }}>Лидер может получить специальное разрешение от Куратора на увольнение сотрудников без объяснения причин (например, для полной расчистки состава).</span>
