***

* **Добавлены префиксы в таблицу фракций.**
* **Обновление пункта A-8 правил лидеров:**
  * **A-8** — «День блата»: <span style={{ color: '#ff0000' }}>в первые 24 часа срока лидер может назначать любые ранги (например, смена лидеров подразделений или заместителей).</span> <span style={{ color: '#00be00' }}>В первые</span> **<span style={{ color: '#00be00' }}>24 часа</span>** <span style={{ color: '#00be00' }}>с момента назначения на пост, лидер имеет право свободно распределять любые ранги внутри фракции (назначение заместителей, глав подразделений и т.д.).</span>
    * **<span style={{ color: '#00be00' }}>A-8.1</span>** <span style={{ color: '#00be00' }}>— В этот период разрешается принимать неограниченное количество новых сотрудников на любые должности без официального принятия (правило</span> **<span style={{ color: '#00be00' }}>B-7</span>** <span style={{ color: '#00be00' }}>).</span>
    * **<span style={{ color: '#00be00' }}>A-8.2</span>** <span style={{ color: '#00be00' }}>— Для стандартного увольнения или понижения игрока по-прежнему требуется веская законная причина (РП-составляющая (</span>**<span style={{ color: '#00be00' }}>B-7</span>**<span style={{ color: '#00be00' }}>)).</span>
    * **<span style={{ color: '#00be00' }}>A-8.3</span>** <span style={{ color: '#00be00' }}>— В «День блата» лидер может получить специальное</span> **<span style={{ color: '#00be00' }}>разрешение от Куратора</span>** <span style={{ color: '#00be00' }}>организации на увольнение сотрудников без объяснения законных причин (например, для полной расчистки состава под новую команду).</span>
* **Обновление правила B-6:**
  * **B-6** — <span style={{ color: '#ff0000' }}>Основания для повышения: успешное прохождение испытательного срока, положительная аттестация/сдача экзаменов, отсутствие взысканий, особые заслуги, решение Менеджмента.</span> <span style={{ color: '#00be00' }}>Повышение присваивается исключительно за обоснованные профессиональные достижения (успешное прохождение испытательного срока, аттестация, сдача экзаменов или особые заслуги, подтвержденные результатом); при этом формальное «решение руководства» не является самостоятельным поводом для роста без конкретного вклада сотрудника, а наличие бана полностью исключает возможность повышения.</span>
* **Обновление правила B-7:**
  * **B-7** — <span style={{ color: '#ff0000' }}>Основания для понижения/увольнения: нарушение дисциплины, систематическое неисполнение обязанностей, проступки, по собственному желанию, решение Менеджмента.</span> <span style={{ color: '#00be00' }}>Любое принудительное понижение или увольнение должно иметь четкую законную игровую причину, подтвержденную доказательствами.</span>
    * **<span style={{ color: '#00be00' }}>B-7.1</span>** <span style={{ color: '#00be00' }}>— Сотрудник имеет право покинуть фракцию по собственному желанию. В этом случае увольнение фиксируется без необходимости предоставления доказательств нарушений.</span>
    * **<span style={{ color: '#00be00' }}>B-7.2</span>** <span style={{ color: '#00be00' }}>— В случае грубого нарушения игроком правил сервера или неадекватного поведения, снятие с должности может быть произведено по решению</span> **<span style={{ color: '#00be00' }}>Менеджмента или Куратора</span>** <span style={{ color: '#00be00' }}>фракции (вне зависимости от наличия игровых причин).</span>
