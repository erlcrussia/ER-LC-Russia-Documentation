    * **A-4** — Прежде чем надевать на кого-либо наручники, вы должны отыграть это.
      * **Пример:** «Я достаю наручники из подсумка и фиксирую ваши руки за спиной».
        **Действие без отыгровок в сцене/канале считается <span style={{ color: '#ff0000' }}>Fail RP</span><span style={{ color: '#00be00' }}>Non RP</span>.**
    * **A-11** — Каждый игрок обязан находиться в Discord-сцене во время игры. Никнейм в Discord должен совпадать с игровым. Использование программ для изменения голоса или игнорирование обращений в голосовом канале приравнивается к <span style={{ color: '#ff0000' }}>Fail RP</span><span style={{ color: '#00be00' }}>Non RP</span>.

  <Accordion title="Обновление Правил Фракций">
    <div className="flex justify-center items-center w-full -mt-10"><h3 className="text-3xl font-bold"><strong>B. Работа с Кадрами</strong></h3></div>

    * **B-6.2** — В случае грубого нарушения игроком правил сервера или неадекватного поведения, снятие с должности может быть произведено <span style={{ color: '#00be00' }}>по решению</span> **<span style={{ color: '#00be00' }}>Менеджмента или Куратора</span>** <span style={{ color: '#00be00' }}>фракции (вне зависимости от наличия игровых причин).</span>
    * **<span style={{ color: '#00be00' }}>B-7 — Процедура кадровых решений:</span>**
      * **<span style={{ color: '#00be00' }}>B-7.1 —</span>** <span style={{ color: '#00be00' }}>Высший состав назначается лидером фракции с обязательным уведомлением Менеджмента.</span>
      * **<span style={{ color: '#00be00' }}>B-7.2 —</span>** <span style={{ color: '#00be00' }}>Понижение за нарушение должно сопровождаться ссылкой на основание; сотруднику даётся возможность дать объяснения, кроме случаев грубого нарушения, требующего немедленного увольнения.</span>
    * **<span style={{ color: '#00be00' }}>B-8 — Все кадровые изменения фиксируются через команды бота «Управление Департаментами MRP».</span>**

    <div className="flex justify-center items-center w-full -mt-14"><h3 className="text-3xl font-bold"><strong>B. Работа с Кадрами — Мультифракция</strong></h3></div>

    * **B-9** (ранее B-11) — Мультифракция и совмещение должностей.
      * **B-9.1 —** Игрок обязан выполнять полный объем установленных квот (норму смен/отчетов) в каждой из выбранных фракций.
      * **Запрещено находиться в этих фракциях одновременно:**
        * <span style={{ color: '#ff0000' }}>СК РФ и любая другая фракция.</span>
        * <span style={{ color: '#ff0000' }}>Прокуратура РФ и любая другая фракция.</span>
        * <span style={{ color: '#ff0000' }}>Суд и любая другая фракция.</span>
        * <span style={{ color: '#00be00' }}>Правительство (СК, Прокуратура, ФСО) и любая другая фракция.</span>
        * МВД и ФСБ.
        * ОПГ и МВД.
        * ОПГ и ФСБ.
        * <span style={{ color: '#ff0000' }}>ОПГ и СК РФ.</span>
        * <span style={{ color: '#ff0000' }}>ОПГ и Суд.</span>
        * <span style={{ color: '#ff0000' }}>ОПГ и Прокуратура РФ.</span><span style={{ color: '#00be00' }}>ОПГ и Правительство (СК, Прокуратура, ФСО).</span>
        * <span style={{ color: '#ff0000' }}>ФСО и ОПГ.</span>

    <div className="flex justify-center items-center w-full -mt-14"><h3 className="text-3xl font-bold"><strong>C. Исполнительная Дисциплина</strong></h3></div>

    * **C-5** — Все плановые <span style={{ color: '#00be00' }}>и внеплановые</span> проверки прокуратуры должны быть заранее согласованы с Менеджером Фракций.
    * **C-6.2** — Если представитель Менеджмента лично участвовал в процессе, решение о его пересмотре или отмене принимается исключительно Высшим менеджментом (<span style={{ color: '#ff0000' }}>Владельцем и Совладельцами</span><span style={{ color: '#00be00' }}>Владельцем и Руководством</span>) <span style={{ color: '#ff0000' }}>Вердикт Высшего менеджмента является окончательным и обжалованию не подлежит</span><span style={{ color: '#00be00' }}>и их решение уже не подлежит обжалованию</span>.

    <div className="flex justify-center items-center w-full -mt-14"><h3 className="text-3xl font-bold"><strong>F. Особенная Часть</strong></h3></div>

    * Разделы G-M (МВД, ФСБ, МЧС, МЗ, Прокуратура, СК, Суд) объединены в раздел **F. Особенная Часть**.
    * **<span style={{ color: '#00be00' }}>F-1 —</span>** <span style={{ color: '#00be00' }}>Право на проведение проверок деятельности государственных и частных структур имеют органы Прокуратуры. В отношении частных организаций (бизнеса) полномочиями по осуществлению проверочных мероприятий также наделяются сотрудники Мэрии.</span>
    * **<span style={{ color: '#00be00' }}>F-2 —</span>** <span style={{ color: '#00be00' }}>Нельзя задерживать и дооформлять игрока, который уже умер в ходе перестрелки и оказался в больнице — требуется новая IC‑ситуация.</span>
    * **<span style={{ color: '#00be00' }}>F-3 —</span>** <span style={{ color: '#00be00' }}>По факту официального обращения в органы ФСБ (подачи тикета на сервере ФСБ) и/или возбуждения уголовного дела другие сотрудники государственных структур РФ могут быть задержаны или привлечены к ответственности органами ФСБ, с участием следователей и дальнейших процедур судо- и делопроизводства.</span>
    * **<span style={{ color: '#00be00' }}>F-4 —</span>** <span style={{ color: '#00be00' }}>Прокуратура РФ имеет право осуществлять надзорную деятельность в отношении государственных фракций и частных организаций.</span>
    * **<span style={{ color: '#00be00' }}>F-5 —</span>** <span style={{ color: '#00be00' }}>Сотрудники СК на месте преступления обладают особыми полномочиями и вправе вызывать сотрудников МВД.</span>
    * **<span style={{ color: '#00be00' }}>F-6 —</span>** <span style={{ color: '#00be00' }}>Сотрудники ФСО имеют право организовывать работу других ведомств при осуществлении охранных мероприятий объекта.</span>
    * **<span style={{ color: '#00be00' }}>F-7 —</span>** <span style={{ color: '#00be00' }}>Фракции имеют права на осуществление и иной деятельности, предусмотренной законодательством РФ, но не нарушающей правила проекта.</span>
  </Accordion>
