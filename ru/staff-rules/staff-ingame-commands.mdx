<div className="flex justify-center items-center w-full -mt-10">
  <h3 className="text-3xl font-bold">Команды Для Модерации+</h3>
</div>

1. `:tocar` — Позволяет переместиться к своему авто. ( работает только на машине)
2. `:toatv` — Позволяет переместиться к своему багги, газонокосилке или квадроциклу. ( работает только на мотовездоходе)
3. `:logs` — Позволяет посмотреть логи сервера.
4. `:kl` — Позволяет посмотреть список убийств на сервере.
5. `:h` — Позволяет оповестить весь сервер.
6. `:pm player` — Позволяет отправить человеку личное сообщение.
7. `:m` — Позволяет отправить всем людям на сервере личное сообщение.
8. `:tp player` — Позволяет переместить выбранного человека к себе.
9. `:tp player(кого) player(к кому)` — Позволяет переместить выбранного человека к другому игроку.
10. `:bring player` — Позволяет переместить к себе выбранного игрока.
11. `:to player` — Позволяет переместиться к выбранному игроку.
12. `:kill player` — Позволяет убить игрока.
13. `:kick player(or id)` — Позволяет кикнуть человека с сервера.
14. `:heal player` — Позволяет восстановить HP игроку или вернуть его к жизни.
15. `:time 00 - 23` — Позволяет установить время на сервере.
    - К примеру: `:time 12` ( Позволяет установить 12 часов дня на карте)
16. `:wanted player` — Позволяет выдать розыск игроку.
17. `:unwanted player` — Позволяет убрать розыск игроку.
18. `:cmds` — Позволяет игроку просмотреть список команд.
19. `:refresh player` — Позволяет перезагрузить скин игрока.
20. `:pt number` — Данная команда позволяет сделать режим в котором нельзя наносить другим урон.
    - К примеру: `:pt 60` ( создается режим на 60 секунд в котором нельзя никого бить и убивать)
21. `:prty` — Данная команда позволяет сделать приоритетный режим на какое-то количество времени.
    - К примеру: `:prty 60` ( создается приоритетный режим на 60 секунд).
22. `:bans` — Данная команда позволяет просмотреть баны
23. `:helpers` — Данная команда позволяет просмотреть помощников на этом сервере.
24. `:mods` — Данная команда позволяет просмотреть модераторов на этом сервере.
25. `:admins` — Данная команда позволяет просмотреть админов на этом сервере.
26. `:startnearfire string` — Данная команда позволяет начать где-то поблизости пожар.
    - `:stopdumpsterfire` — Позволяет прекратить пожар в мусорном баке.
    - `:stopfire string` — Позволяет закончить пожар
    - `:starfire string` — Позволяет начать пожар

<div className="flex justify-center items-center w-full -mt-14">
  <h3 className="text-3xl font-bold">Команды Для Администрации+</h3>
</div>

1. `:loadlayout string` — Команда для Map Editor. Позволяет загрузить ту или иную карту, чтобы не искать ее вручную.
2. `:unloadlayout string` — Команда для Map Editor. Позволяет убрать ту или иную загрузку карты.
3. `:shutdown` — Команда создана для перезагрузки сервера. Во время этого, все игроки будут кикнуты с сервера.
   <Danger>
     **Использовать данную команду можно только с разрешением Владельца.**
   </Danger>
4. `:weather` — Команда позволяет человеку сделать или убрать погоду. **К примеру:**
   - `:weather fog` — Создает на сервере туман.
   - `:weather snow` — Создает на сервере снег.
   - `:weather thunderstorm` — Создает на сервере грозу.
   - `:weather rain` — Создает на сервере дождь.
   - `:weather clear` — Убирает на сервере погоду.
5. `:mod player(or id)` — Позволяет выдать человеку роль "Модератор" на сервере.
6. `:unmod player(or id)` — Позволяет отнять у человека роль "Модератор" на сервере.
7. `:helper player(or id)` — Позволяет выдать человеку роль "помощник" на сервере.
8. `:unhelper player(or id)` — Позволяет отнять у человека роль "помощник" на сервере.
9. `:ban player(or id)` — Позволяет забанить человека на сервере.
10. `:unban player(or id)` — Позволяет разбанить человека на сервере.
